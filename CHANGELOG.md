# Changelog

所有显著变化会记录在此文件。版本遵循语义化版本约定；Preview 版本可能调整本机数据格式和兼容策略。

## [1.5.0-preview.1] - 2026-07-22

### Added

- 任意数量的隔离 Profile、动态两列卡片 Grid，以及逐 Profile 启动与运行状态跟踪。
- ChatGPT 账号、OpenAI API Key、第三方 Responses Provider 三种认证方式。
- 每个 Profile 独立且持久化的主题色，并应用到卡片和托盘角标运行变体。
- 通用工作空间首次配置向导，支持新建、导入和旧 profile 一次性迁移。
- HTTPS 与回环 HTTP Base URL 校验，拒绝 URL 内嵌凭据。
- API Key 密码输入、隔离原子写入，以及快照/日志排除验证。
- 可注入临时路径的隔离测试，覆盖导入白名单、旧 profile 发现和个人目录不变。
- 公开仓库文档、安全策略、隐私说明、CI 和仓库审计脚本。

### Changed

- 旧单例注册与进程状态改为一次性迁移；新版运行只使用 `profiles.json` 和 Profile 进程字典。
- 发现多个旧 marker 时全部原地迁移，不再要求从中选择唯一工作空间。
- 界面从固定组织通道改为用户自定义“工作空间”。
- 运行时自检不再扫描硬编码源码目录；仓库敏感信息审计改为独立脚本和 CI。
- 源码版本升级到 `1.5.0-preview.1`。

### Security

- 导入排除任务、SQLite、日志、插件缓存、系统 Skills、重解析点和常见凭据文件。
- 兼容旧 profile 时只登记原目录，不移动或重写其配置。
